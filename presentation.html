02 — The Protocol

What is UCP?

Universal Commerce Protocol — One standard for all commerce platforms

❌ Without UCP

  • Every AI agent needs custom integration per platform
  • SAP OCC, Shopify API, Magento REST — all different
  • N platforms × M agents = N×M integrations
  • No standard discovery — need manual configuration

✔ With UCP

  • One standard API works across all platforms
  • Single protocol: discovery, checkout, orders, identity
  • N + M integrations (each implements UCP once)
  • /.well-known/ucp — universal discovery
🔍

Discovery

AI agents find store capabilities, payment methods, and API endpoints via a single URL

🛒

Checkout

Create sessions, add items, set shipping, complete orders through standardized flow

📦

Orders

Track status, get tracking numbers and delivery updates

🔒

Identity

OAuth2-based authentication and user linking

05 — Checkout Flow

UCP Checkout Session

How an AI agent completes a purchase through UCP API calls

1

Create Session

Empty cart created with unique session ID

POST /ucp/api/checkout/sessions
2

Add Items

Products added by code + quantity, pricing recalculated

PATCH /ucp/api/checkout/sessions/{id}
3

Set Shipping Address

Delivery address attached to session

PATCH /ucp/api/checkout/sessions/{id}
4

Set Delivery Method

Standard $5.99 / Express $14.99 / Overnight $24.99

PATCH /ucp/api/checkout/sessions/{id}
5

Place Order

Cart converted to order with tracking number

POST /ucp/api/checkout/sessions/{id}/complete

incomplete → ready → complete

07 — API Reference

All Endpoints

MethodEndpointDescription
GET /.well-known/ucp Merchant UCP discovery profile
POST /ucp/api/checkout/sessions Create new checkout session

GET /ucp/api/orders/{id} Get order status + tracking

POST /ucp/api/identity/token Exchange code for access token
GET /ucp/api/identity/userinfo Get authenticated user info
POST /api/chat Send message to AI assistant
GET /api/chat/status Check Gemini AI connection
POST /api/chat/reset Clear conversation history
